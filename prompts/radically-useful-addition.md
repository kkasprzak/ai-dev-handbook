# Radically Useful Addition

## Purpose

Use this prompt to challenge an existing development plan or project and surface one high-leverage improvement that may not be obvious yet.

## Use When

- You think a development plan is already complete.
- You have a fleshed-out project and want a fresh strategic improvement idea.
- You want to compare suggestions from a few different frontier models.

## Context To Provide

- The current development plan or project description.
- The goal, constraints, and intended users.
- Any implementation decisions that are already fixed.
- Whether the assistant should focus on product value, architecture, developer workflow, UX, business impact, or another dimension.

## Prompt

```text
What's the single smartest and most radically innovative and accretive and useful and compelling addition you could make to the plan at this point?
```

For an already developed or fleshed-out project, use this variant:

```text
What's the single smartest and most radically innovative and accretive and useful and compelling addition you could make to the project at this point?
```
